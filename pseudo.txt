Writing pseudocode for FizzBuzz

No user interface. web console program

User will enter data (popup box)

desired output - list of numbers from 1 to the number user entered but each number that is divisible by 3 will output Fizz, each number divisible by 5 will output
Buzz, each number divisible by both 3 and 5 will output FizzBuzz


When a user inputs a number
Loop from 1 to the entered number
if current number is divisible by 3 then print "Fizz"
if current number is divisible by 5 then print "Buzz"
if current number is divisible by both 3 and 5 print "FizzBuzz"
otherwise print current number
